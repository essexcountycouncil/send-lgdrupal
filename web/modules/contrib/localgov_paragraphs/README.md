# LocalGovDrupal Paragraphs

Core paragraph components for the LocalGovDrupal distribution.

Contains the following paragraph types:

* Advanced links (advanced_links)
* Contact (contact)
* Image (pagebuilder_image)
* Text (pagebuilder_text)
